# gamelog 기획서 — "셋로그의 게임 버전"

> 작성일: 2026-06-06 · 전제: `docs/01-setlog-역기획.md`의 setlog 코어 루프를 게이머 맥락으로 이식

---

## 0. 한 줄 컨셉

**"오늘 뭐 했어?"가 아니라 "오늘 뭐 했어, 게임에서?"** — 친구들과 게임하는 순간을 초단위 클립으로 기록하면, 하루가 끝날 때 우리 파티의 '오늘의 게임 브이로그'가 자동으로 완성되는 프라이빗 게이머 소셜 앱.

setlog가 '일상'을 동시성으로 묶었다면, gamelog는 **'플레이 세션'**을 묶는다. 게이머는 이미 같은 시간에 같은 행동(게임)을 하는 집단이라 setlog의 핵심 가설(강제된 동시성)이 자연 발생적으로 성립하는 시장이다.

---

## 1. 방향성 5가지

### 방향 1. 데일리 게임 브이로그 (setlog 직계 이식) ⭐ 코어
- **무엇**: 게임 세션 중 알림/버튼 트리거로 2~5초 클립(화면+음성 or 캠) 기록 → 하루 끝에 친구들과 분할화면 "오늘의 겜방일지" 자동 합성
- **왜**: setlog로 검증된 루프를 그대로 활용. 게이머의 하루는 일반인의 하루보다 콘텐츠 밀도가 높음 (승리, 사망, 레전드 순간)
- **차이점**: 정시 알림 대신 **세션 기반 트리거** — "게임 시작했네? 한 판 끝나면 찍어!" (알림 피로 문제 해결)

### 방향 2. 순간 포착 — 라이트 하이라이트 클립
- **무엇**: 플레이 중 "방금 그거!" 버튼 한 번으로 직전 N초 클립 저장 → 그룹 피드 공유 (Medal.tv의 소셜·라이트 버전)
- **왜**: 게임 콘텐츠의 보상 밀도를 극대화. 무편집 원칙과 궁합 좋음
- **리스크**: 모바일 화면 상시 녹화는 OS 제약·배터리 이슈 → PC 컴패니언 또는 '직후 15초 재연 촬영'으로 우회

### 방향 3. 게이머 BeReal — "지금 뭐 해?" 동시 접속 인증
- **무엇**: 알림이 오면 지금 하는 게임 화면 스샷 + 전면캠 셀카 동시 촬영 → 친구들이 서로 뭘 하는지 실시간 확인 → "나도 그거 켤게" 합류 버튼
- **왜**: 게이머의 만성 문제 "같이 할 사람 없음(LFG)"을 소셜 그래프로 해결. 동시성 인증이 곧 파티 모집이 됨

### 방향 4. 자동 게임 다이어리 + 리캡 (솔로 가치 보강)
- **무엇**: 게임 계정 연동(Steam/Riot/PSN 등)으로 플레이 기록 자동 수집 + 한 줄 감상 → 주간/월간/연간 자동 리캡 영상 ("게임판 Spotify Wrapped")
- **왜**: setlog 최대 약점인 '그룹 깨지면 앱도 죽는' 구조 보완. 솔로여도 쌓이는 가치 = 리텐션 바닥 방어. 연말 리캡은 외부 공유 바이럴 폭발력 큼

### 방향 5. 데일리 퀘스트 — 같은 미션, 다른 플레이
- **무엇**: 매일 그룹에 공통 미션 발급 ("오늘 1승 클립 인증", "안 해본 캐릭으로 한 판", "최고 데미지 순간") → 각자 인증 → 분할화면 비교 영상
- **왜**: setlog의 '강제된 동시성'을 '강제된 동일 과제'로 변형. 매일 다른 미션 = 콘텐츠 다양성 문제(setlog 약점 2) 해결 + 게이머에게 익숙한 퀘스트 문법

### 방향성 종합 — 제품 전략

```
코어 루프   = 방향 1 (검증된 setlog 루프의 게임 이식)
훅 강화     = 방향 5 (데일리 퀘스트로 매일 새로운 이유 제공)
리텐션 바닥 = 방향 4 (솔로 다이어리·리캡으로 그룹 붕괴 방어)
확장 단계   = 방향 3 (LFG 소셜 그래프) → 방향 2 (본격 클립 인프라)
```

## 2. 타깃 사용자

| 페르소나 | 설명 | 핵심 욕구 |
|---|---|---|
| 🎮 듀오/파티 게이머 (1차) | 매일 밤 디스코드 켜고 같이 하는 18-29세 친구 그룹 | "우리끼리 논 거 남기고 싶다, 근데 편집은 귀찮다" |
| 📱 모바일 게이머 | 출퇴근길 같은 게임 하는 친구들 | "쟤 아직도 그 게임 하네 ㅋㅋ" 가벼운 연결감 |
| 📔 기록형 솔로 게이머 | 플레이 이력을 모으고 싶은 사람 | 내 게임 인생의 아카이브, 연말 결산 |

## 3. 코어 루프 상세

```
[트리거]                  [액션]                    [보상]                      [투자]
친구가 게임 시작 알림      한 판 끝날 때마다          친구들의 오늘 플레이 클립    파티 로그 축적,
+ 데일리 퀘스트 발급   →   2~5초 클립 + 한 줄 캡션 →  + 23시 "오늘의 파티 브이로그" →  스트릭, 시즌 리캡,
("ㅇㅇ님이 롤 켰어요")     (퀘스트 인증 겸용)         자동 합성 + 리액션/DM        퀘스트 연승 기록
```

- **사이클**: setlog의 4AM 리셋 대신 **저녁 게임 타임 중심(18시~새벽 4시)** 사이클. 합성 영상은 다음날 정오 발행 (새벽 게이머 포함)
- **무편집 원칙 유지**: 찍은 그대로, 사후 업로드 불가 → "현질 인증, 정신승리, 발컨까지 날것 그대로"가 매력
- **분할화면 합성**: 같은 게임이면 같은 순간끼리 정렬 (우리 팀 한타를 4명 시점으로)

## 4. MVP 기능 명세 (P0)

| # | 기능 | 설명 |
|---|---|---|
| 1 | 파티(그룹) 생성 | 2~6명 초대, 솔로 로그 가능 |
| 2 | 세션 클립 촬영 | 카메라(캠/화면) 2~5초 + 캡션 + 게임 태그 |
| 3 | 데일리 퀘스트 | 매일 1개 공통 미션, 인증 시 스트릭 +1 |
| 4 | 자동 브이로그 합성 | 사이클 종료 시 시간순/분할화면 합성, 게임 태그·스코어 오버레이 |
| 5 | 리액션/DM | 클립별 이모지 리액션 + 그룹 채팅 |
| 6 | 저장/공유 | 워터마크 포함 세로 영상 내보내기 (릴스/틱톡/쇼츠 규격) |
| 7 | 플레이 상태 알림 | "친구가 게임 시작" 알림 (수동 체크인 → 추후 계정 연동 자동화) |

**P1 (MVP+1)**: 게임 계정 연동(Riot/Steam) 자동 전적 오버레이 · 주간 리캡 · 파티 통계("이번 주 우리 27판")
**P2**: 연간 Wrapped · LFG 합류 버튼 · PC 컴패니언(자동 클립 캡처) · 게임별 공개 커뮤니티(선택적)

## 5. 차별화 요약 (vs setlog / Medal / BeReal)

| | setlog | Medal.tv | gamelog |
|---|---|---|---|
| 대상 | 일상 | 게임 클립(공개) | **게임하는 친구 그룹(프라이빗)** |
| 트리거 | 정시 알림 | 수동/자동 캡처 | **세션·퀘스트 기반** (알림 피로↓) |
| 결과물 | 하루 브이로그 | 단일 클립 | **파티 단위 합성 브이로그 + 리캡** |
| 솔로 가치 | 약함 | 보통 | **자동 다이어리·Wrapped로 강함** |

## 6. 성장 & 수익

- **성장**: ① 파티 초대 = 활성화(K-factor 내장) ② 합성 영상 외부 공유 루프 ③ 연말 Wrapped 시즌 바이럴 ④ 게임 커뮤니티(롤/발로란트 클랜, 디스코드 서버) 시드 마케팅
- **수익(후순위)**: 프리미엄 합성 테마/오버레이, 고화질·장기 아카이브 구독, (장기) 게임사 제휴 퀘스트(신작 미션 = 네이티브 광고)

## 7. 리스크

| 리스크 | 대응 |
|---|---|
| 모바일 화면 녹화 제약 (iOS ReplayKit 동의 필요, 백그라운드 불가) | MVP는 캠/사진+짧은 화면녹화 수동 방식, 자동 캡처는 PC 컴패니언으로 |
| 그룹 붕괴 = 이탈 (setlog 동일 약점) | 솔로 다이어리·리캡(방향 4)으로 바닥 방어 |
| 게임사 약관/저작권 (게임 화면 사용) | 사적 그룹 공유는 통상 허용 범위, 외부 공유 시 UGC 정책 게임 우선 지원 |
| setlog가 게임 기능 추가할 가능성 | 게임 태그·전적 연동·퀘스트 등 도메인 깊이로 선점 |

## 8. 검증 로드맵 (12주)

1. **W1-2**: 프로토타입 없이 검증 — 디스코드 봇으로 데일리 퀘스트+클립 인증 실험 (코어 루프 가설 테스트)
2. **W3-8**: MVP 개발 (P0 7개 기능, iOS 우선 — setlog와 동일 전략)
3. **W9-12**: 클로즈드 베타 — 대학 게임 동아리·디스코드 서버 10곳, 측정 지표: **D7 파티 생존율, 일일 클립 수/인, 외부 공유율**
